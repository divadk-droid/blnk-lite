# BLNK Brand Design Specification

## Logo Requirements

### Primary Logo
- **Size:** 512x512px (square)
- **Format:** PNG (transparent), SVG (vector)
- **Style:** Modern, tech, security-focused

### Design Elements
| Element | Specification |
|---------|--------------|
| Icon | Shield or lock + circuit pattern |
| Colors | Primary #3B82F6 (blue), Accent #10B981 (green) |
| Typography | Sans-serif, bold |
| Feel | Trust, security, AI, futuristic |

### Variants Needed
1. **Full Logo:** Icon + "BLNK" text
2. **Icon Only:** For favicon, small sizes
3. **Monochrome:** White/black versions
4. **Inverse:** For dark/light backgrounds

## Banner Requirements

### Hero Banner
- **Size:** 1920x400px
- **Format:** PNG, JPG
- **Use:** Website header, Twitter header

### Design Elements
| Element | Specification |
|---------|--------------|
| Background | Gradient (dark blue to purple) |
| Pattern | Subtle circuit/grid lines |
| Text | "Secure AI Agents" or tagline |
| Logo Placement | Left side, 20% padding |

### Variants Needed
1. **Website Banner:** 1920x400
2. **Twitter Header:** 1500x500
3. **Discord Banner:** 1920x1080
4. **Thumbnail:** 1200x630 (OpenGraph)
